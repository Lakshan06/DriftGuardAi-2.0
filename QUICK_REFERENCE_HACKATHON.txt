KEY ALGORITHMS
================================================================================

1. POPULATION STABILITY INDEX (PSI)
   Purpose: Detect feature distribution changes
   Formula: Î£ (actual_% - expected_%) Ã— ln(actual_% / expected_%)
   Threshold: PSI â‰¥ 0.25 = DRIFT DETECTED
   Implementation: NumPy histogram-based

2. KOLMOGOROV-SMIRNOV (KS) TEST
   Purpose: Compare cumulative distributions
   Formula: max |F_actual(x) - F_expected(x)|
   Threshold: KS â‰¥ 0.20 = DRIFT DETECTED
   Implementation: scipy.stats.ks_2samp

3. DISPARITY SCORE (Fairness)
   Purpose: Detect bias across demographic groups
   Formula: max(approval_rate) - min(approval_rate)
   Threshold: disparity > 0.10 = BIAS DETECTED
   Implementation: Group-based approval rate comparison

4. MODEL RISK INDEX (MRI)
   Purpose: Composite risk score (0-100)
   Formula: (drift_component Ã— 0.60) + (fairness_component Ã— 0.40)
   Output: 0-30 Green, 30-60 Yellow, 60-80 Orange, 80+ Red
